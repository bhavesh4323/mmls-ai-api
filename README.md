# MMLS-AI v2.0 — Military Medical Logistics System

AI-powered inventory management for military medical supplies.

**Built by:** Bhavesh | AI Intern @ Black Space Technology  
**Tech Stack:** Python · XGBoost · LSTM · Flask · MongoDB · Streamlit · Plotly

## Features
- 📦 Real-time inventory with 5-tier Hours-of-Supply engine
- 🤖 XGBoost demand forecasting (R² = 0.9937)
- 📈 LSTM MASCAL surge detection
- 🔍 Medicine search with FDA database lookup
- 🗺️ Cross-echelon supply chain map
- 📋 Controlled drug audit log
